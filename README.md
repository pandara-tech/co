# Pandara Tech

Smart solutions for smart applications.

## Structure

- `files/` — Website source (HTML, CSS, JS, images, fonts, legal docs)
- `files/index.html` — Main landing page with splash, about, products, why, contact
- `files/mentis.html` — Hidden Mentis product page (no public links)
- `files/eula.html` — End User License Agreement (EN/AR)
- `files/privacy-policy.html` — Privacy Policy (EN/AR)
- `files/terms-of-use.html` — Terms of Use (EN/AR)

## Features

- Arabic/English language toggle
- RTL support with BoutrosDinkum font
- Azonix font for splash title
- Telegram bot contact form
- Full-screen sections with scroll snap
